Estudo em Lote - Cálculo de Custo de Empregados

Como usar o modelo de cálculo em lote

Campos obrigatórios

  • Nome: Nome completo do empregado
  • Salário: Valor bruto mensal (ex: 5000)
  • Tipo de Contrato: clt_normal, aprendiz, estagiario ou pj

Campos opcionais

  • Diária: 0, 668.80, 900.00, 1776.50, 2821.50
  • Adicional: SIM ou NÃO (exatamente assim)
  • Dependentes: 0, 1, 2, 3... (números inteiros)

Dicas importantes

  • Não altere os cabeçalhos das colunas
  • Use vírgula como separador decimal (não ponto)
  • Mantenha as opções exatamente como mostrado (SIM/NÃO, tipos de contrato)
  • Não deixe células vazias, use 0 quando não houver valor
  • Salve o arquivo em formato .xlsx
